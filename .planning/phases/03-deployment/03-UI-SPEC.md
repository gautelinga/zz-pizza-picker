## New Element Contract: "Menu last updated" Date

### Element specification

| Property | Value |
|----------|-------|
| Element type | Static `<p>` in `index.astro` body (NOT a Svelte component) |
| Rationale | Build-time data; no reactivity needed; Svelte island unnecessary |
| Position | Footer of picker section, below the `<PizzaPicker>` island |
| Tailwind classes | `text-sm text-neutral-400 text-center` |
| Container | Wrap in `<div class="w-full max-w-[400px] mx-auto px-4 pb-8">` to match picker section width |

### Content

| Property | Value |
|----------|-------|
| Template | `Menu last updated: {day} {Month} {year}` |
| Example | `Menu last updated: 7 Apr 2026` |
| Day format | No leading zero (7, not 07) |
| Month format | 3-letter English abbreviation, title case (Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec) |
| Year format | 4-digit full year |
| Data source | `scraped_at` field from `public/menu.json` (ISO-8601 string, e.g. `"2026-04-07T12:00:00.000Z"`) |
| Parsing location | `index.astro` frontmatter — parse and format before rendering |

### Formatting implementation

```
// index.astro frontmatter
const scrapedAt = new Date(menu.scraped_at);
const day = scrapedAt.getUTCDate();                          // no leading zero
const month = scrapedAt.toLocaleString('en-GB', { month: 'short', timeZone: 'UTC' });
const year = scrapedAt.getUTCFullYear();
const lastUpdated = `Menu last updated: ${day} ${month} ${year}`;
```

Render as: `<p class="text-sm text-neutral-400 text-center">{lastUpdated}</p>`
